# Twitter Thread Skill

Generates a full-length Persian thread (5–15 tweets) with strong hook, structured body, and CTA close.

## Inputs
- Topic or source research/post file
- Desired length: short (5–7) | medium (8–12) | long (13–15)

## Thread Structure

**Tweet 1 — Hook**
- Most surprising or bold claim
- Ends with "🧵" to signal thread
- Example pattern: `[Bold claim]. [Why you should care]. 🧵`

**Tweets 2–N — Body**
- One idea per tweet, fully self-contained
- Use numbered format: `2/`
- Each tweet should be valuable on its own (shareable standalone)
- Mix: facts, examples, "did you know?", tools, tips

**Second-to-last tweet — Summary**
- "خلاصه بگم:" + 3–5 bullet points of key takeaways

**Last tweet — CTA**
- "اگه مفید بود، ریتوییت کن تا بقیه هم ببینن ♻️"
- Or: follow + save prompt

## Persian Standards
- Colloquial throughout
- Short sentences — Twitter punishes walls of text
- Number each tweet clearly: `1/`, `2/`, etc.

## Use via
`/write-post [topic] twitter thread`
