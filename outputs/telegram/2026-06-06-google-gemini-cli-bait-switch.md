<b>گوگل ۶۰۰۰ کمک رایگان گرفت، بعد اون آدم‌ها رو از ابزارشون بیرون انداخت</b>

ابزار Gemini CLI گوگل، یه دستیار AI برای ترمینال برنامه‌نویسا، بیشتر از ۱۰۰ هزار ستاره گیتهاب داشت.

هزاران نفر ازش استفاده کردن، باگ گزارش دادن، و بیشتر از ۶۰۰۰ تا pull request رایگان کد بهش فرستادن.

گوگل هم توی اعلامیه‌ی رسمیش از همین کمک‌ها اسم برد و تشکر کرد.
بعد اعلام کرد از ۱۸ ژوئن، ابزار فقط برای مشتریان enterprise باز می‌مونه.

جایگزین برای بقیه؟ ابزار Antigravity CLI.
که تو گیتهابش الان فقط یه readme و یه GIF دمو هست. سورس کد نداره.

کسایی که ابزار رو ساختن، نه می‌تونن ازش استفاده کنن، نه می‌تونن جایگزینش رو حتی بررسی کنن.

شاید باید قبل از اینکه روی ابزار «رایگان» شرکت‌های بزرگ بیلد کنیم، بپرسیم این رایگانی تا کِی می‌مونه. 👀
