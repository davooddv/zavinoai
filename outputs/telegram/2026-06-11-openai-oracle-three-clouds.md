<b>شرکت OpenAI الان روی سه Cloud بزرگ دنیا در دسترسه</b>

شرکت Oracle، یکی از بزرگ‌ترین غول‌های Cloud دنیا، از دیروز اعلام کرد که مشتریانش می‌تونن از همون اعتبار Cloud موجودشون برای مدل‌های شرکت OpenAI هم استفاده کنن.

یعنی اگه سازمانی قرارداد Oracle داره، نیازی به ثبت حساب جداگانه با OpenAI نداره.
بودجه‌ای که قبلاً فقط برای زیرساخت بود، الان می‌تونه برای GPT-5.5 و ابزار Codex هم خرج بشه.

جالب‌تر اینکه چند هفته پیش، دقیقاً همین اتفاق روی AWS آمازون افتاد.
یعنی شرکت OpenAI الان روی سه Cloud بزرگ دنیا در دسترسه: Azure مایکروسافت، AWS آمازون، و Oracle.

چیزی که یه سال پیش انحصاری مایکروسافت Azure بود،
الان داره تبدیل می‌شه به زیرساخت مشترک کل صنعت Cloud. 🤖
