<b>مدل Claude الان توی اکسل کار می‌کنه برای ۷۵۰ میلیون نفر</b>

امروز مایکروسافت اعلام کرد مدل Claude از شرکت Anthropic به Excel Agent Mode اضافه شد.

اکسل، نرم‌افزاری که ۷۵۰ میلیون نفر در دنیا ازش استفاده می‌کنن، الان می‌تونه یه مدل frontier AI رو مستقیم داخل خودش اجرا کنه.

حالا می‌تونی از Claude بخوای داخل همون اکسل فرمول بنویسه، داده‌های نامرتب رو پاکسازی کنه، از محتوای جدول گزارش بسازه، یا workflow خودکار راه بندازه؛ بدون اینکه از هیچ ابزار جانبی کمک بگیری.

جالب‌تر اینکه Claude قبل از این هم توی Copilot، Azure، و Microsoft 365 بود. اما ۷۵۰ میلیون کاربر اکسل بزرگ‌ترین سطح توزیعیه که هر مدل AI تا الان بهش رسیده.

چیزی که قبلاً فقط توی ابزارهای تخصصی AI می‌دیدیم، الان داخل همون فایل اکسلیه که سال‌هاست ازش استفاده می‌کنی. 🚀
