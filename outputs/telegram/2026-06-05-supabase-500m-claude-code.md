<b>بزرگ‌ترین مشتری Supabase یه انسان نیست؛ ابزار Claude Code هست 🤖</b>

شرکت Supabase، پلتفرم پایگاه‌داده متن‌باز، تازه ۵۰۰ میلیون دلار جمع کرد و ارزشش به ۱۰.۵ میلیارد دلار رسید.
این عدد در کمتر از ۸ ماه تقریباً دو برابر شده.

دلیل اصلیش اینه که ابزارهای کدنویسی AI مثل Claude Code و Codex هر بار که یه اپ می‌سازن، یه پایگاه داده هم نیاز دارن.
و Supabase همون پایگاه داده‌ای‌ه که AI ازش استفاده می‌کنه.

جالب‌تر اینکه بخش مربوط به پلتفرم‌ها در ۶ ماه گذشته ۳۷۰٪ رشد کرده.
نه به خاطر توسعه‌دهنده‌های بیشتر، بلکه چون AI‌ها خودشون دارن برنامه می‌نویسن.

الان ۹ میلیون توسعه‌دهنده از این پلتفرم استفاده می‌کنن.
ولی برای اولین بار، یه ابزار AI بیشتر از هر انسانی داره پایگاه داده می‌سازه.

موج vibe coding داره یه اقتصاد موازی می‌سازه.
شرکت‌هایی که زیرساخت AI هستن، ساکت‌ترین برنده‌های این دورانن. 📈
