<b>مایکروسافت ۱۳ میلیارد دلار توی OpenAI ریخت، حالا داره جاشو می‌گیره 🤖</b>

شرکت مایکروسافت، بزرگ‌ترین سرمایه‌گذار OpenAI، داره یه مدل هوش مصنوعی کاملاً داخلی می‌سازه تا GPT رو از داخل محصولاتش بیرون کنه.

اسمش پروژه Polaris هست. از آگوست ۲۰۲۶، این مدل به عنوان موتور پیش‌فرض ابزار GitHub Copilot جایگزین GPT-4 Turbo میشه.

تغییر مهم دیگه‌ای هم هست. ابزار GitHub Copilot دیگه «همکار برنامه‌نویس» نیست، الان «همتای برنامه‌نویس» شده. یعنی می‌تونه مستقل باگ بگیره، کد بنویسه، و pull request باز کنه. بدون اینکه کسی بهش بگه.

همزمان، شرکت مایکروسافت یه Windows Agent Store هم راه‌اندازی کرد. سهم درآمد توسعه‌دهنده‌ها؟ ۸۵٪. در مقایسه، App Store اپل ۷۰٪ میده.

بزرگ‌ترین سرمایه‌گذار OpenAI داره محصولاتش رو طوری بازطراحی می‌کنه که کمتر به OpenAI نیاز داشته باشه. توی دنیای AI، حتی شریک‌ها هم برای خودشون بازی می‌کنن. 📈
