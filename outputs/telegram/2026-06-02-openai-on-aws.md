<b>شرکتی که پشتش مایکروسافته، الان روی سرورهای Amazon هم هست 👀</b>

شرکت OpenAI، سازنده‌ی ChatGPT، تا همین اواخر انحصاراً به پلتفرم Azure مایکروسافت متکی بود. حالا مدل‌هاشو به صورت رسمی روی AWS از شرکت Amazon هم عرضه کرده.

مدل‌های GPT-5.5 و GPT-5.4، ابزار Codex و سرویس Managed Agents، الان همه روی Amazon Bedrock در دسترسن.

ابزار Codex هر هفته بیشتر از ۴ میلیون کاربر فعال داره. این تیم‌ها الان می‌تونن از همون ابزار داخل زیرساخت AWS خودشون استفاده کنن، بدون اینکه چیزی رو از سیستم‌شون بیرون ببرن یا تنظیمات امنیتی رو از نو بچینن.

جالب‌تر اینکه مایکروسافت بزرگ‌ترین رقیب ابری Amazon هست. حالا شرکت OpenAI که مایکروسافت ۱۳ میلیارد دلار توش ریخته، داره روی سرورهای رقیب هم اجرا میشه.

قبلاً برای دسترسی به مدل‌های OpenAI تو enterprise، فقط یه مسیر وجود داشت: Azure. حالا دو تا داره. 📈
