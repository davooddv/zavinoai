<b>یه مدل رایگان چینی ادعا می‌کنه از قوی‌ترین Claude بهتره 🤖</b>

شرکت Moonshot AI از چین، مدل Kimi K2.7 Code رو با ۱ تریلیون پارامتر منتشر کرد.

وزن‌هاش روی Hugging Face رایگانه. می‌شه روی سرور خودت اجراش کنی. بدون اشتراک، بدون هزینه ماهانه، بدون دردسر.

جالب‌تر اینکه این مدل ادعا می‌کنه روی معیار tool-use از Claude Fable 5 Opus، قوی‌ترین مدل آنتروپیک، بهتر عمل می‌کنه.

همین هفته‌ای که آنتروپیک فرم بورس پر کرد و Claude رو مرکز استراتژی‌اش قرار داد.

چیزی که قبلاً فقط شرکت‌های میلیارد دلاری داشتن، الان روی یه سرور معمولی اجرا می‌شه. 🚀
