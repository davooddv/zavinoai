<b>یه توسعه‌دهنده حساب کرد قبضش از ۲۹ دلار میشه ۷۵۰ دلار</b> 👀

ابزار GitHub Copilot، دستیار کدنویسی شرکت مایکروسافت، از همین امروز مدل قیمت‌گذاریش رو تغییر داد.

قبلاً ماهانه ۱۰ دلار می‌دادی، بدون محدودیت استفاده می‌کردی.
حالا باید به ازای هر توکنی که مصرف می‌کنی پول بدی.

یه توسعه‌دهنده محاسبه کرد قبضش از ۲۹ دلار ماهانه قراره بشه ۷۵۰ دلار.
یکی دیگه از ۵۰ دلار به ۳ هزار دلار.

پست رسمی اعلام این تغییر در فروم GitHub تا الان ۹۰۴ دیسلایک گرفته. در مقابل ۲۲ لایک.

جالب‌تر اینکه خود شرکت توضیح داد که هزینه‌های عملیاتی Copilot از اول امسال هر هفته تقریباً دو برابر می‌شده.
یعنی هزینه‌ی واقعی استفاده از AI داره به‌قدری بالا میره که حتی مایکروسافت هم دیگه نمی‌تونه مدل فلت ماهانه رو حفظ کنه. 📈
