<b>مایکروسافت ۱۳ میلیارد دلار رو تو OpenAI ریخت، حالا داره رقیبش رو می‌سازه</b>

شرکت Microsoft، همون شرکتی که از ۲۰۱۹ تا الان ۱۳ میلیارد دلار تو OpenAI سرمایه‌گذاری کرده، تازه تو کنفرانس Build 2026 یه خانواده مدل AI معرفی کرد به اسم MAI که هدفش اینه که دیگه به OpenAI نیازی نداشته باشه.

مدل کدنویسی این خانواده روی معیار SWE-bench با مدل Claude 3.7 Sonnet شرکت Anthropic برابری می‌کنه، با هزینه‌ای خیلی پایین‌تر. یعنی ابزار GitHub Copilot از الان می‌تونه بدون اینکه یه تا API از OpenAI صدا کنه، کار کنه.

جالب‌تر اینکه مدل تبدیل صدا به متن Microsoft، به اسم MAI-Transcribe-1.5، الان رتبه اول جهان داره و از مدل Whisper شرکت OpenAI هم جلو زده.

شراکتی که چند سال پیش معماری کل صنعت AI رو شکل داد، داره آروم از هم می‌پاشه. ساکت‌ترین جدایی‌ها معمولاً مهم‌ترین‌ان. 🎯
