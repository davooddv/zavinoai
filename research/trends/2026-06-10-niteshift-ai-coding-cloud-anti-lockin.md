# Niteshift: The Cloud Built for AI Coding Agents — Not Developers

## What Is It
Two former Datadog engineers launched Niteshift, a cloud platform built specifically for AI coding agents (like Claude Code, Codex, and open-source models). Instead of running AI coding tools locally or inside a specific vendor's platform, Niteshift runs agents in fully configured cloud environments, routes between different AI models based on the task, and produces pull requests with verification artifacts — all triggered from Slack, Linear, or GitHub.

## Key Facts & Stats
- Seed round raised: $7 million — Source: TechCrunch, June 10, 2026
- Lead investor: Greylock (Jerry Chen)
- Other investors: Amplify Partners, BoxGroup, SV Angel
- Notable angels: Reid Hoffman, Datadog founders Olivier Pomel and Alexis Le-Quoc
- Founders: Sajid Mehmood and Conor Branagan (nearly a decade at Datadog each)
- Platform is now generally available (launched alongside funding announcement)
- Supports Claude Code, Codex, and open-source models — routes between them automatically

## Why It Matters for ZAVINO's Audience
As AI coding agents become standard, the question shifts from "which AI should I use?" to "where do I run it and how do I manage it?" Niteshift represents a new category: infrastructure for AI agents, not for humans. Persian-speaking developers and tech-oriented followers watching the vibe coding trend will find this relevant — it's the plumbing layer that makes multi-agent coding workflows possible.

## The Surprising Angle
The founders' explicit bet is against Big AI lock-in. Every major AI company (OpenAI with Codex, Anthropic with Claude Code, GitHub with Copilot) wants you to run your coding agents inside their ecosystem. Niteshift says: run them all, route between them, own the infrastructure yourself. It's the "we don't care which AI wins, we care that AI coding keeps running" play.

The secondary angle: this is a small $7M seed round, but the backers (Reid Hoffman, Greylock, Datadog founders) are not small. The signal from who believes in it matters more than the round size.
