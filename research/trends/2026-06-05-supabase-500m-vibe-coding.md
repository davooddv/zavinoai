# Supabase — $500M, $10.5B Valuation, Riding the AI Coding Wave

## What Is It
Supabase، یه پلتفرم دیتابیس متن‌باز، روز ۴ ژوئن ۲۰۲۶ موفق شد ۵۰۰ میلیون دلار جذب کنه و ارزش‌گذاریش به ۱۰.۵ میلیارد دلار رسید. دلیل اصلی رشد؟ ابزارهای AI coding مثل Claude Code و Codex — که الان مسئول اکثریت دیتابیس‌های جدید روی پلتفرم Supabase هستن.

## Key Facts & Stats
- مبلغ جذب‌شده: ۵۰۰ میلیون دلار (Series F)
- ارزش‌گذاری: ۱۰.۵ میلیارد دلار — تقریباً ۲ برابر دور قبلی (اکتبر ۲۰۲۵)
- سرمایه‌گذاران: GIC (هدایت‌کننده)، Accel، Y Combinator، Coatue، Stripe
- کاربران: ۹ میلیون توسعه‌دهنده در سراسر جهان، ۲۵۰ هزار مشتری پرداختی
- تعداد کارمندان: ۳۵۰ نفر
- رشد ۶ ماهه: بخش Supabase for Platforms طی ۶ ماه گذشته ۳۷۰ درصد رشد کرد
- Claude Code بزرگ‌ترین منبع دیتابیس‌های جدید در ۲۰۲۶ است — بیشتر از هر ابزار دیگه‌ای
- تاریخ انتشار: ۴ ژوئن ۲۰۲۶
- منابع: CNBC، SiliconAngle، PRNewswire (همه ۴ ژوئن ۲۰۲۶)

## Why It Matters for ZAVINO's Audience
این خبر یه چیز رو آشکار می‌کنه که خیلی‌ها بهش فکر نکردن: هر بار که Claude Code یا Codex یه اپ می‌سازه، یه دیتابیس هم نیاز داره. Supabase داره از «vibe coding» پول درمیاره — نه با ساختن مدل AI، بلکه با تأمین زیرساختی که مدل‌های AI بهش نیاز دارن. برای توسعه‌دهنده‌های ایرانی این خبر مهمه چون نشون می‌ده چه زیرساخت‌هایی زیر موج AI coding دارن بزرگ می‌شن.

## The Surprising Angle
ابزارهای AI coding الان مسئول بیشتر از نیمی از دیتابیس‌های جدید روی Supabase هستن — نه انسان‌ها. Claude Code به تنهایی بزرگ‌ترین «مشتری» Supabase در ۲۰۲۶ شده. این یعنی AI ها دارن برای هم زیرساخت می‌سازن — بدون اینکه انسانی در حلقه باشه. ارزش‌گذاری شرکت در کمتر از ۸ ماه ۲ برابر شد، فقط به خاطر این موج.
