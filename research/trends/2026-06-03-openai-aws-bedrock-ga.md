# OpenAI روی AWS — GPT-5.5 و Codex در Amazon Bedrock عمومی شدن

## What Is It
از اول ژوئن ۲۰۲۶، مدل‌های GPT-5.5، GPT-5.4، و ابزار Codex از شرکت OpenAI به‌طور کامل در سرویس Amazon Bedrock در دسترس همه‌ی شرکت‌هاست. این یعنی میلیون‌ها کسب‌وکار که از زیرساخت AWS استفاده می‌کنن، الان می‌تونن بدون جابجایی پلتفرم، مستقیم با مدل‌های OpenAI کار کنن.

## Key Facts & Stats
- تاریخ عمومی‌شدن: **۱ ژوئن ۲۰۲۶** (یک ماه بعد از preview محدود در آوریل ۲۰۲۶)
- مدل‌های در دسترس: **GPT-5.5** (بهترین کیفیت) و **GPT-5.4** (قیمت-کیفیت)
- Codex هم همزمان در Bedrock عمومی شد — اما بیشتر از ۴ میلیون توسعه‌دهنده از Codex هفتگی استفاده می‌کنن
- قیمت‌گذاری: **دقیقاً همون نرخ OpenAI** — بدون هزینه اضافی از AWS
- استفاده روی AWS تبدیل به **AWS commitment** می‌شه — برای شرکت‌هایی که قرارداد AWS دارن، می‌تونن این هزینه رو از سهمیه‌شون کسر کنن
- هر تماس API با کنترل‌های امنیتی AWS همراهه: IAM، VPC، KMS encryption، CloudTrail
- موجود در: US East (Ohio) برای GPT-5.5، و US East + US West (Oregon) برای GPT-5.4
- منابع: AWS Official Blog (June 1, 2026)، OpenAI Official (June 1, 2026)

## Why It Matters for ZAVINO's Audience
خیلی از شرکت‌های فناوری و استارتاپ‌های منطقه از AWS استفاده می‌کنن. قبلاً برای استفاده از GPT-5.5 باید سراغ OpenAI مستقیم می‌رفتی — الان می‌تونی همون مدل رو داخل زیرساخت AWS که داری ازش استفاده می‌کنی اجرا کنی. این یعنی امنیت بیشتر، compliance راحت‌تر، و پرداخت از یه جا.

## The Surprising Angle
این قرارداد نشون می‌ده که OpenAI و Amazon — که روزی رقیب بودن (Amazon روی Claude/Anthropic سرمایه‌گذاری کرده) — الان دارن مستقیم با هم کار می‌کنن. GPT-5.5 و مدل‌های Claude هر دو روی AWS Bedrock در دسترسن — یعنی AWS داره به «سوپرمارکت AI» تبدیل می‌شه که مدل‌های رقیب رو کنار هم می‌فروشه.
