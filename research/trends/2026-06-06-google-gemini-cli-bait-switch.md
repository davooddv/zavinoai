# Google Gemini CLI: Accepted 6,000 Free Contributions, Then Locked Developers Out

## What Is It
ابزار Gemini CLI گوگل — یه agent متن‌باز برای برنامه‌نویسا که توی ترمینال کار می‌کنه — بعد از اینکه جامعه توسعه‌دهندگان ۶۰۰۰ pull request رایگان بهش دادن، از ۱۸ ژوئن ۲۰۲۶ برای همه به جز مشتریان enterprise بسته می‌شه. جایگزین (Antigravity CLI) هنوز سورس کد نداره.

## Key Facts & Stats
- Gemini CLI originally released: mid-2025 under Apache 2.0 license (open-source)
- GitHub stars accumulated: 100,000+
- Community contributions: 6,000+ merged pull requests from external developers
- Google announcement date: May 19, 2026
- Free access cutoff date: June 18, 2026
- Who loses access: free users, Google AI Pro/Ultra subscribers, individual Gemini Code Assist users
- Who keeps access: Enterprise customers with paid Gemini Code Assist Standard or Enterprise licenses
- Replacement tool: "Antigravity CLI" — but as of late May 2026, its GitHub repo contains only a changelog, a readme, and a demo GIF. No source code.
- Coverage: TechTimes (2026-05-29), IBTimes (2026-05-23), VPSMAC Blog (2026-06-03), AIBuilderClub

## Why It Matters for ZAVINO's Audience
خیلی از توسعه‌دهندگان ایرانی از Gemini CLI استفاده می‌کردن چون رایگان بود و قدرتمند. این تصمیم نشون می‌ده که حتی ابزارهای open-source گوگل هم در نهایت ممکنه فقط برای enterprise باشن. این یه هشدار جدی برای هر کسی که روی ابزارهای "رایگان" شرکت‌های بزرگ بیلد می‌کنه.

## The Surprising Angle
گوگل ۶۰۰۰ pull request رایگان از جامعه گرفت — یعنی هزاران ساعت کار داوطلبانه — و بعد از اینکه ابزار رو با کمک همین آدم‌ها تکمیل کرد، اون‌ها رو از محصول نهایی بیرون انداخت. جانشینش هنوز سورس کد نداره. این یه case study کلاسیک از "مشارکت جامعه برای محصول enterprise" هست.
